# oop_polygons

This JAVA Program calculate the polygon's area (Triangles & Rectangles) the program it's able to store in an array N Triangles & Rectangles, at the final, the program shows the area & each one's data. 

+ Super class "Polygon"
+ Sub class "Rectangle"
+ Sub class "Triangle"
